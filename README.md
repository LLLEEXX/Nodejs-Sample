# Nodejs-Sample

Simple Signin and Login

## Start the server:

```
npm run startDev
```

The server will run on `http://localhost:3000`

## Endpoints:

- `POST /signup-user` - Sign up a new user
- `POST /login` - Log in a user
- `GET /user-name/:id` - Fetch user name

## Dependencies

- express
- nodemon
- mongoose
- bcrypt
